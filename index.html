Pembuatan Esai Kegiatan Praktikum

Minggu 1 dan 2

Pada Minggu ini praktikum mempelajarai terkait dasar-dasar dari komponen HTML, JavaScript, dan CSS. HTML merupakan bahasa markup standar untuk pembuatan web. CSS berkaitan dengan desain tampilan web. Dalam HTML terdapat komponen elemen dan tag

Minggu 3

Pada minggu ini dilakukan percobaan pembuatan halaman web sederhana yang dapat dilakukan dengan laman W3School. Pada minggu dibuat juga membuat CV sederhana dengan bantuan template dari WIX Studio

Minggu 4

Pada minggu ini dilakukan percobaan pembuatan halaman web sederhana yang berisi fungsi untuk melakukan hitungan statistisk, meliputi validasi password, validasi umur, perhitungan bilangan prima, perhitungan bilangan fibonacci.

Minggu 5

Pada minggu ini dilakukan percobaan pembuatan halaman web pada github dan juga melakukan pembuatan CV dengan Bootstrap. Pada minggu ini juga dilakukan pembuatan landing Page.

Minggu 6

Pada minggu ini dilakukan pembauatan peta online dengan Leaflet JS. Melakukan pembuatan peta rute dari Tugu Yogyakarta ke Alun-Alun Kidul dengan dilengkapi Heatmap.

Minggu 7

Pada minggu ini dilakukan pembuatan peta online dengan Leaflet Js dan GEOJson. Melakukan pembuatan peta administrasi daerah Kabupaten Gunungkidul.
